# テキーラ観覧車 -ShotFerrisWheel-

## 概要

グラスをセットしてルーレットを回し、止まったショットグラスを飲んで楽しめる「テキーラ観覧車」です。  
バーやクラブ、イベント会場、パーティーワールドなどに設置することで、お酒をテーマにしたゲームや交流コンテンツを手軽に導入できます。  
ショットグラスの着脱、ルーレット演出、当たり判定、ゲーム進行を備えており、複数人で気軽に遊べるVRChat向けギミックとしてご利用いただけます。

本商品は VRChat の PC / Quest の両方に対応しています。

## 特徴

- ショットグラスの着脱
- 最大12個のショットグラスに対応
- ルーレット演出
- 当たりグラスのランダム抽選
- グラスごとの状態同期
- PC / Quest 両対応
