# セットアップガイド

このページでは「テキーラ観覧車 -ShotFerrisWheel-」の詳細な導入方法について説明します。

## 動作環境

- Unity 2022.3.22f1
- VRChat SDK - Base 3.10.4 以上
- VRChat SDK - World 3.10.4 以上

## 前提アセット

本商品を使用するには、以下のアセットが必要です：

### Filamented Standard for Unity

**作者**: s-ilent  
**URL**: https://s-ilent.booth.pm/items/3250389

:::warning 重要
このアセットがインポートされていないと、マテリアルが正常に表示されない場合があります。先にインポートしてから本商品をインポートしてください。
:::

## 導入手順

### 1. Unitypackage のインポート

解凍した zip ファイルに同梱されている `unitypackage` を Unity にインポートします。

### 2. Prefab の配置

#### 手順

1. **Project ウィンドウ** で `Assets/TpLab/ShotFerrisWheel/Prefabs` フォルダを開きます
2. 使用したい **Prefab** を選択します
3. **Hierarchy** にドラッグ＆ドロップして配置します

:::tip 初心者の方へ

**ボトル** と **グラス** はオプションです。  
必要に応じて個別に配置して使用してください。
:::

#### 利用可能な Prefab

| カテゴリ      | Prefab 名            | 説明           | 備考                       |
| ------------- | -------------------- | -------------- | -------------------------- |
| 📦 **メイン** | ShotFerrisWheel      | テキーラ観覧車 | テキーラ観覧車本体         |
| 🍾 **ボトル** | Bottle/TequilaBottle | テキーラボトル | ショットグラスと組み合わせ |
| 🥃 **グラス** | Glass/ShotGlass      | ショットグラス | テキーラボトルから注げる   |

### 3. ゲームの設定

TequilaFerrisWheel/Game/GameSettings の Inspector 上で設定を行います。

| 設定項目              | 説明             | 設定値          | 効果                                     |
| --------------------- | ---------------- | --------------- | ---------------------------------------- |
| **Max Hit Count**     | 最大ヒット数     | 数値（例：3）   | ゲーム設定で指定できるヒットするの最大値 |
| **Default Hit Count** | ヒット数の初期値 | 数値（例：1.0） | ゲーム設定のヒット数の初期値             |

## 使用方法

### 基本的な操作手順

#### 🍾 ボトルからグラスに注ぐ

1. **ボトルを手に取る** - VR でボトルを掴みます
2. **Use ボタンを押すと** キャップが外れます
3. **グラスの上で傾ける** と液体が注がれます

#### 🥃 グラスから飲む

1. **グラスを手に取る** - VR でグラスを掴みます
2. **グラスを傾ける** と中身が揺れます
3. **さらに傾ける** と中身が減っていきます

### 視覚効果の向上

ボトルやグラスの光沢を美しく見せるためには、ワールドに **ReflectionProbe** を配置することをお勧めします。

:::tip 参考サイト
Reflection Probe を置いて"映える"ワールドを作ろう  
https://creator.cluster.mu/2022/03/22/misaka-1/
:::
