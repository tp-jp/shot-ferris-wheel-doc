# Support & Contact

This page provides support information and contact methods for "Tequila Ferris Wheel".

### Compatibility

- Old version prefabs and new version prefabs may not be compatible
- Be sure to create a backup before updating

### If Not Working

1. Check your **Unity version** (2022.3.22f1)
2. Check your **VRChat SDK version** (3.10.4 or later)
3. Check for **console errors**
